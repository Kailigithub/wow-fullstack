亲爱的朋友，欢迎来到Vue3的学习园地！

- - 第1.4节-样式（30分钟）
- - 第1.5节-表单（45分钟）
- 第2章-数据（共计180分钟）

- - 第2.5节-列表渲染（60分钟）
- - 第2.6节-侦听（30分钟）
- - 第2.7节-存储（30分钟）
- 第3章-事件（共计60分钟）
